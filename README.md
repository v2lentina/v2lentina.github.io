## Highlights

- Dark and light theme toggle

- About section with external links

## Tech Stack

- React + TypeScript
- Vite
- CSS (custom styling)

## Getting Started

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Deployment

This repo is set up for GitHub Pages via GitHub Actions. The workflow builds the site and deploys the `dist` folder automatically on push to `main`.
